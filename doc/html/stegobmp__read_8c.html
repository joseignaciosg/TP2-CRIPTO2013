Extract a hidden encrypted file from a BMP file using LSB1 algorithm.

Parameters:
imagea file stream to a BMPv3 file
msg_fa pointer to a file stream that is not open yet for the output
namea pointer to a string containing the base name of the output file
passwdthe password to be used to decrypt the extracted file
algothe cipher algorithm to be used to decrypt the extracted file
blk_algothe cipher chaining mode of operation to be used to decrypt the extracted file
Returns:
0 if a file could be succesfully extracted, -1 otherwise
