int load_img_header ( FILE *  in,
struct bmp_type *  img 
)

Load an BMPv3 header from a file stream into a bmp_type struct.

This function receives a correctly allocated bmp_type img and sets its fields with the information from the first 54 bytes readable from in. It left the field img->matrix untouched (so it doesn't have to be allocated).

Parameters:
inthe file stream corresponding to an open BMPv3 file
imgan allocated bmp_type structure
int load_img_matrix ( FILE *  in,
struct bmp_type *  img 
)

Load the matrix of pixels from a BMPv3 file into a bmp_type struct.

This functions receives a correctly allocated bmp_type img with an allocated img->matrix field and loads the byte corresponding to the image content from the file stream in to img.

Parameters:
inthe file stream corresponding to an open BMPv3 file
imgan allocated bmp_type structure with an allocated matrix field
Warning:
This function doesn't make any checking on in. It's the responsability of the programmer to load first the header in img and performs the appropriate checking.
